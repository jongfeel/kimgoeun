### javascript vs css

- javascript: 동적인 것들, user와 상호작용하는 것들, event, alert창
- css: 위치, 디자인, 레이아웃 잡는 역할
- javascript로 css의 요소를 컨트롤 할 수 있음
- css: 어느 정도 동적인 요소, 이벤트 (mousehover, click), 산수 범위내에서 계산과 계산식을 통해서 동적 제어가 가능
